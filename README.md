# Omakase-Sansyoku
完全ランダムな色を3色を出して、デザインツールにコピペしやすくしました。

■ アプリの概要
「三色カラーパレット」は、ランダムに3つの色組みを生成し、直感的に配色を確認できるツールです。
直感的なUIで、生成されたカラーコードの管理やお気に入りの配色の保存が簡単に行えます。
デザイン制作やイラストの色選び、配色のアイデア出しにお役立てください。

■ 主な機能と使い方
1. 新しい色を生成
   「新しい色を生成」ボタンを押すと、ランダムに3つの色の組み合わせが画面に表示されます。

2. このカラーを保存
   現在表示されているお気に入りの3色を保存できます。
   最大30件まで保存可能で、30件を超えると古いものから上書きされます。

3. 履歴機能
   - 【保存履歴を見る】: 保存したお気に入りの色組みを一覧で確認できます。
   - 【生成履歴を見る】: 過去に生成された色組みの履歴（最大30件）を確認できます。生成履歴から「保存」を選択してお気に入りに登録することも可能です。

4. コピー機能（履歴画面）
   履歴画面のカラーチップ横にあるボタンから、以下の操作が可能です。
   - 【画像コピー】: 3色の帯状の画像をクリップボードにコピーします。画像ソフトなどにそのまま貼り付けてスポイトで色を取るのに便利です。
   - 【WEBカラーコピー】: 3色分のWEB用カラーコード（例: #FFFFFF, #000000...）をまとめてクリップボードにコピーします。
   - 【右クリックコピー】: マウスを色チップに乗せ、特定の色の上で右クリックすると、その1色だけのWEBカラーコードをコピーできます。

■ 動作環境
Windows / macOS / Linux

■ 導入とアンインストール
・Windowsの場合 (.exe): 
  ダウンロードした実行ファイル（おまかせ三色.exe）をダブルクリックするだけで起動します。
・macOSの場合 (.app / .dmg):
  ダウンロードしたアプリ（おまかせ三色.app）を「アプリケーション」フォルダに移動して起動してください。
  ※初回起動時にセキュリティの警告が出る場合は、「システム設定」>「プライバシーとセキュリティ」から「このまま開く」を選択してください。
・Linuxの場合 (実行権限付きバイナリ):
  ダウンロードしたファイルにターミナル等で実行権限を付与し（例: `chmod +x omakase_sanshoku`）、ダブルクリックまたはターミナルから実行してください。

・アンインストール: 
  フォルダごと削除してください。（レジストリ等は使用していません）
  ※起動すると、同じフォルダ内に `reference` というフォルダが自動生成され、そこに保存履歴のデータ（.json形式）が保存されます。こちらも本体と一緒に削除して構いません。

■ Linuxをご利用の方へ（画像コピー機能について）
Linux環境でカラーチップの「画像コピー」機能をお使いになる場合、あらかじめOS側にクリップボード操作用のコマンドラインツール（`xclip` または `wl-copy`）がインストールされている必要があります。コピー時にエラーが出る場合は、いずれかのツールを導入してください。
